# Tourgether UAT (User Acceptance Testing) 시나리오

## 개요
**목적:** 운영 환경 배포 전 전체 기능 검증  
**대상 플랫폼:** Web (Desktop/Mobile 반응형)  
**지원 언어:** 한국어(ko), 영어(en), 일본어(ja), 중국어(zh), 프랑스어(fr), 스페인어(es)  
**테스트 환경:** Staging 환경 (운영 DB 미러링)
