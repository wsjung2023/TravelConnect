📦 Tourgether 소스코드 다운로드

다운로드 가능한 파일들:

최신 소스코드: 파일 업로드 시스템 완료 버전

📥 소스코드 다운로드 (99.6MB) 📥 전체 백업 다운로드 (604MB) - 업로드된 파일 포함

🔧 설치 방법:

  1. 위 파일 중 하나를 다운로드
  2. 압축 해제: tar -xzf 파일명.tar.gz
  3. npm install 실행
  4. .env 파일 생성하여 환경변수 설정
  5. npm run dev로 서버 시작

주요 환경변수:

DATABASE_URL=postgresql://...
VITE_GOOGLE_MAPS_API_KEY=your_api_key
