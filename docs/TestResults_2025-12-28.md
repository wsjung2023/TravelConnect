## 테스트 원칙

1. **대원칙 준수**
   - 코드 주석: 한국어
   - API 응답: camelCase
   - 마스터 데이터: DB 기반
   - 화면 이동: 버튼 진입 방식 (URL 직접 입력 금지)

2. **결제 테스트 정책**
   - 모든 결제: USD(달러)로만
   - 사람이 해야 하는 결제 부분: 패스/우회
   - 별도 수동 테스트 항목으로 기록

3. **실시간 상호작용**
   - WebSocket 연결 상태
   - 알림 실시간 수신
   - 채팅 메시지 전달

4. **시스템 퍼포먼스**
   - API 응답 시간
   - 페이지 로딩 속도
   - 데이터베이스 쿼리 성능
