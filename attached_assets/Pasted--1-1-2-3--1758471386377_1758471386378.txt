시나리오 1: 새 사용자 여행 콘텐츠 작성 여정
1. [인증] 사용자 로그인/회원가입
2. [홈페이지] 메인 지도 화면 로드 확인
3. [포스트 작성] 지도에서 위치 클릭 → 포스트 작성 모달 열기
4. [파일 업로드] 여행 사진 업로드 (MIME 타입 검증)
5. [콘텐츠 입력] 제목, 내용, 테마 선택
6. [저장] 포스트 저장 → 지도에 마커 생성 확인
7. [피드 확인] 메인 피드에 포스트 표시 확인
시나리오 2: 소셜 상호작용 테스트
1. [피드 탐색] 다른 사용자 포스트 보기
2. [좋아요] 포스트 좋아요 → 카운트 증가 확인
3. [댓글] 댓글 작성 → 실시간 업데이트 확인
4. [지도 연동] 포스트의 지도 마커 클릭 → 상세 정보 표시
5. [필터링] 테마별, 지역별 포스트 필터링 기능
시나리오 3: 실시간 채팅 시스템
1. [채널 목록] 채팅 채널 리스트 로드
2. [메시지 전송] 새 메시지 작성 → 실시간 전달 확인
3. [스레드 기능] 메시지에 스레드 답글 → 3-panel 동작 확인
4. [WebSocket 안정성] 연결 끊김 → 자동 재연결 테스트
5. [읽음 상태] 메시지 읽음 처리 → 채널 목록 업데이트
시나리오 4: 지도 통합 기능
1. [마커 표시] 포스트 위치 마커 정확도 확인
2. [클러스터링] 줌 레벨별 마커 클러스터링
3. [POI 연동] Google Places API 연동 확인
4. [위치 검색] 장소 검색 → 지도 이동 기능
5. [테마별 색상] 포스트 테마에 따른 마커 색상 구분
시나리오 5: 사용자 프로필 & "만나요" 기능
1. [프로필 설정] 사용자 정보 수정
2. [Open to Meet] 만나요 기능 토글 ON/OFF
3. [지역 설정] 현재 지역 설정 → 근처 사용자 목록 확인
4. [자동 만료] 만나요 상태 자동 만료 동작 확인
시나리오 6: 관리자 기능 (기존 완성)
1. [관리자 로그인] 관리자 권한 확인
2. [포스트 관리] 사용자 포스트 조회/삭제
3. [시스템 관리] 데이터베이스 관리 도구 접근
4. [법적 문서] 약관/정책 페이지 관리
🔍 테스트 우선순위
P0 (필수):

시나리오 1, 2, 3 (핵심 SNS 기능)
P1 (중요):

시나리오 4, 5 (지도, 사용자 상호작용)
P2 (보조):

시나리오 6 (관리자 기능)

💼 시나리오 7: 유튜버/개인 가이드 ↔ 여행자 SNS 거래
[가이드 측]
1. [프로필 설정] 유튜버/가이드로 프로필 설정 (전문가 인증 뱃지)
2. [경험 등록] 특별한 여행 경험/투어 포스트 작성
   - 예: "서울 숨은 맛집 투어", "부산 야경 포토스팟 가이드"
3. [가격 설정] 경험별 가격 및 가능 일정 설정
4. [포트폴리오] 기존 가이드 영상/후기 링크 첨부
[여행자 측]
5. [가이드 발견] 지역별/테마별 가이드 검색
6. [포트폴리오 확인] 가이드의 SNS 피드, 후기 확인
7. [문의하기] 가이드에게 DM으로 세부사항 문의
8. [예약 요청] 날짜/인원 선택하여 예약 요청
9. [확정 대기] 가이드 승인 대기 → 확정 알림
[매칭 완료 후]
10. [실시간 소통] 전용 채팅방에서 일정 조율
11. [위치 공유] 만날 장소 지도에 핀 고정
12. [후기 작성] 경험 후 상호 리뷰 작성
🛍️ 시나리오 8: 현지인 구매대행 서비스
[구매대행자 (현지 여행자/거주자)]
1. [대행 서비스 등록] "OO지역 구매대행 가능" 포스트 작성
2. [전문 분야 표시] 화장품, 의류, 음식, 기념품 등 카테고리 설정
3. [수수료 공개] 대행비, 배송비 정보 투명하게 공개
4. [신뢰도 구축] 대행 경험, 인증샷 포스트로 신뢰성 어필
[구매 요청자]
5. [대행자 찾기] 현지별 구매대행 서비스 검색
6. [상품 문의] 원하는 상품 사진/링크 전송
7. [견적 요청] 상품 가격 + 대행비 견적 확인
8. [구매 의뢰] 구매 확정 및 결제 (임시 보관)
[대행 과정]
9. [구매 확인] 대행자가 상품 구매 후 인증샷 전송
10. [품질 체크] 상품 상태 사진/영상으로 확인
11. [배송 처리] 국제배송 or 직접 전달 방식 선택
12. [거래 완료] 상품 수령 확인 후 대금 정산
13. [평가 시스템] 상호 평점 및 후기 작성
🎉 시나리오 9: 여행지 이벤트 초대 & 참여
[이벤트 주최자]
1. [이벤트 생성] "오늘 저녁 홍대 펍크롤 함께해요!" 이벤트 포스트
2. [상세 정보] 시간, 장소, 참가비, 최대인원 설정
3. [지도 연동] 모임 장소 정확한 위치 핀 표시
4. [참가자 모집] SNS 피드를 통한 실시간 모집
[참가 희망자]
5. [이벤트 발견] 현재 위치 기반 근처 이벤트 알림
6. [상세 확인] 주최자 프로필, 이벤트 내용 검토
7. [참가 신청] 원클릭 참가 신청 + 간단 자기소개
8. [승인 대기] 주최자 승인 → 참가 확정
[이벤트 당일]
9. [실시간 채팅] 참가자 전용 그룹 채팅방 자동 생성
10. [위치 공유] 실시간 위치 공유로 쉬운 만남
11. [라이브 업데이트] 이벤트 진행 상황 실시간 포스팅
12. [네트워킹] 참가자들 간 연락처 교환, 친구 추가
[이벤트 후]
13. [추억 공유] 이벤트 사진/영상 공동 앨범 생성
14. [후속 모임] 성공적인 이벤트 → 정기 모임 제안
15. [평가 & 신뢰도] 주최자/참가자 상호 평가
🔗 시나리오 간 연계 효과
Cross-Platform 시너지:
1. [SNS → 거래] 일반 여행 포스트 → 가이드 서비스 발견
2. [가이드 → 구매대행] 가이드가 구매대행도 겸업
3. [이벤트 → 장기 관계] 일회성 이벤트 → 개인 가이드 관계 발전
4. [신뢰도 통합] 모든 활동의 평점이 통합 신뢰도 점수로 관리
